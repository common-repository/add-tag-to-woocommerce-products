== Description ==

Add Custom Tag to WooCommerce Products. Like Newest Products, On Sale.

You Can define number of days to display tags and Tag will expire after that Period. 

You can add custom tag text on products from admin panel

If You need any Help Simple <a href="http://themelocation.com/contact/">Contact us</a>

== Installation ==

1. From the Dashboard:
1. Plugins
1. Add New
1. Search "Display Tag Products Woocommerce"
1. Install Now
1. Activate
1. Go to Settings -> Tag Products Woocommerce
